Create a Catkin workspace¶

  1. Step 1. mkdir -p path_to_catkin_ws/src

  2. Step 2. cd path_to_catkin_ws/src

  3. Step 3. git clone https://github.com/hrii-iit/xsens_mvn_ros.git –recursive

#. Step 4. Build your catkin workspace cd .. catkin build
